### 3. 核心写作模型 (`gaokao-full-mark-essay-writing-model.md`)

**"审-立-构-材-语"五维一体模型**:
- 审题 (决定60分生死线)
- 立意 (决定55分突破线)
- 结构 (决定文体专业度)
- 素材 (决定文化含量与厚度)
- 语言 (决定文采与可读性)

### 4. 高考满分作文写作Skill (`gaokao-writing-skill/`)

**七阶段闭环工作流**:
1. **审题** (Topic Comprehension) - 5个立意+最佳立意+候选标题
2. **查找资料** (Resource Gathering) - 古今中外+时政+名句
3. **写提纲** (Outline Draft) - 5-7段完整提纲
4. **核对修改提纲** (Outline Review) - 五查清单
5. **写全文** (Full-Text Writing) - 800-1000字
6. **特级阅卷老师评分** (Examiner Scoring) - 5维度精确打分
7. **按意见修改+再次评分** (Iterative Revision) - 直到55+

## 使用指南

### 1. 备考学习

- 顺序阅读: 10年真题 → 10年范文 → 写作模型 → Skill
- 重点: 范文的"立意角度""结构模式""素材组合"
- 训练: 每天1篇"审题+立意", 每周1篇"完整写作"

### 2. 实战训练

- 启动Skill: 提供一个真题 + "请写一篇55+的作文"
- 跟随7阶段: 审-查-构-核-写-评-改
- 直到分数 ≥ 55: 接受, 记录; < 55: 修改, 再评分

### 3. 教学使用

- 教师: 借助Skill评分模块, 给学生作文快速打分
- 学生: 借助Skill写作模型, 自评互评作文
- 家长: 借助Skill流程, 辅导孩子写作

## 评分标准

| 等级 | 分数 | 标志 |
|------|------|------|
| 一类上 | 56-60 | 满分水准, 极少见 |
| 一类下 | 54-55 | 优秀, 可上重点大学 |
| 二类上 | 50-53 | 良好, 重点线 |
| 二类中 | 48-49 | 中等偏上, 一本线 |
| 二类下 | 45-47 | 中等, 二本线 |
| 三类及以下 | < 45 | 较弱 |

**Skill目标分数**: **55+** (一类下到一类上)

## 重要说明

1. **本项目数据均来自公开渠道**, 仅用于教育、学习、研究目的
2. **2025年部分范文为"高分示范文"**, 由教育部考试中心或阅卷组专家撰写, 非考生作品
3. **范文版权属原作者**, 本项目仅作"教育使用"之"合理使用"
4. **建议批判性使用**, 重点学习"立意深度""结构层次""文化积累"等核心要素
5. **核心方法**: 真正的"写作高手", 不是"考试高手", 而是"人生高手" — 所有的"好作文", 都是"好人生"的注脚
