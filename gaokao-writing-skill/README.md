## Skill概述

本Skill遵循"**审题-查找资料-写提纲-核对修改提纲-写全文-高考语文特级阅卷老师评分提修改意见-按意见修改-再次评分-直到评分在55分以上**"的完整流程, 帮助用户系统化地写出55分以上(满分60分)的高考语文作文。

## 核心特性

1. **七阶段闭环工作流** — 不跳过任何环节, 严格按"审-查-构-核-写-评-改"循环
2. **特级阅卷老师模拟评分** — 嵌入式评分, 5维度(审/立/构/材/语)精确打分
3. **迭代优化** — 5次循环, 每次修改都有具体可执行意见, 直到55+
4. **数据驱动** — 基于近10年(2016-2025)高考真题+满分作文归纳
5. **写作模型支撑** — 配套"审-立-构-材-语"五维一体核心写作模型

## 输出格式

Skill将按7阶段输出:
1. **审题brief** — 包含5个立意+最佳立意+候选标题
2. **资料包** — 古今中外+时政+名句
3. **提纲** — 5-7段完整提纲
4. **核对修改后的提纲** — 含修改理由
5. **全文** — 800-1000字
6. **评分报告** — 5维度打分+修改意见
7. **修改记录** — 每次修改的before/after

## 评分标准

| 分数 | 等级 | 标志 |
|------|------|------|
| 58-60 | 一类上 | 优秀, 罕见 |
| 54-57 | 一类下 | 优秀, 可上重点大学 |
| 50-53 | 二类上 | 良好 |
| 48-49 | 二类中 | 中等偏上 |
| 45-47 | 二类下 | 中等 |
| < 45 | 三类及以下 | 较弱 |

**目标分数**: 55+ (一类下到一类上)

## 阅卷"采分点"

1. **审题** (10分) - 完全扣题, 不偏题
2. **立意** (15分) - 深刻有思辨
3. **内容** (15分) - 古今中外四维
4. **结构** (10分) - 起承转合, 详略得当
5. **语言** (10分) - 流畅有文采, 金句有力
