# Gaokao Essay Prompt Bank (2016-2025)

> This is a working bank of real Gaokao Chinese essay prompts from 2016 to 2025.
> Use this file when the user wants to practice on a real prompt, or when the skill
> needs to choose a representative prompt for the workflow demo.

## How to use this file

1. Pick a year that matches the user's needs (e.g., most recent year, or a specific theme).
2. Copy the prompt verbatim into the Stage 1 审题 brief.
3. Optionally cross-reference `sample-essays.md` for the corresponding year to see the
   reference 满分 essay.

## 2016

### 全国 I 卷

> 阅读下面的漫画材料，根据要求写一篇不少于800字的文章。
> 漫画由两幅图组成：
> - 第一幅图：一位男孩得了100分，被亲了一口；另一位得了55分，挨了一巴掌。

### 全国 II 卷
