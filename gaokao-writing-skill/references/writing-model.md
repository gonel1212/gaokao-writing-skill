# Gaokao Writing — 5-Dimension Model Reference

> Companion reference to SKILL.md. Contains the detailed 审-立-构-材-语 (审题-立意-结构-素材-语言)
> 5-dimension writing model, derived from 10 years of Gaokao essay analysis (2016-2025).

## When to open this file

Open this file when the skill is in **Stage 1, Stage 3, or Stage 6** of the workflow
and you need the underlying model details — not the procedural instructions.

For the procedural workflow, see `SKILL.md`.
For specific prompts to write on, see `references/prompt-bank.md`.
For sample full-mark essays, see `references/sample-essays.md`.

## The 5-dimension model at a glance

| 维度 | 权重 | 核心问题 | 失分原因 |
|------|------|----------|----------|
| 审题 (审) | 决定生死 | 我是否抓到了材料的核心立意? | 偏题、跑题、套作 |
| 立意 (立) | 决定上限 | 我的立意是否深刻有思辨? | 肤浅、伪辩证、套话 |
| 结构 (构) | 决定可读性 | 我的文章是否起承转合? | 散乱、平铺、虎头蛇尾 |
| 素材 (材) | 决定厚度 | 我的论据是否古今中外? | 单薄、雷同、空洞 |
| 语言 (语) | 决定文采 | 我的文字是否流畅有韵? | 病句、套话、网络化 |

## 维度1: 审题 (Topic Comprehension)

### 五步审题法

1. **通读材料，标记关键概念** — 逐字读, 不跳读
2. **拆分结构** — 开头 / 展开 / 结尾 各讲什么
3. **分析关系** — 一元? 二元? 多元?
4. **定位任务** — 文体 / 字数 / 身份 / 情境
5. **形成立意词组** — 3-5 个关键词

### 三原则

- **不跳读** — 漫画要看画中物, 不只看标题
- **不臆断** — 不用自己准备好的模板硬套
- **不先入为主** — 认真读, 别被"押题"带偏

### 一元 / 二元 / 多元材料

| 类型 | 例子 | 审题要点 |
|------|------|----------|
| 一元 | 2023新高考I"故事的力量" | 抓核心: "力量" 的多元含义 |
| 二元 | 2017全国I"中国关键词" | 抓组合: 选2-3个关键词并解释关联 |
| 多元 | 2022新高考I"本手/妙手/俗手" | 抓转化: 三者之间的辩证关系 |

### 任务驱动识别

- 演讲稿 — 开头有称呼, 中间有号召, 结尾有倡议
- 书信体 — 有抬头有落款, 符合收信人身份
- 发言稿 — 有"我发言的题目是……"
- 观后感 — 必有"观"+"感"两部分
- 慰问信 — 必有时间节点代入

## 维度2: 立意 (Central Claim)

### 三层境界

1. **浅层** — 就事论事, 45分以下
2. **中层** — 就事论理, 50-55分
3. **深层** — 就文化/哲学/时代, 55-60分

### 五向思维

- **正向** — 材料说了什么
- **反向** — 材料没说什么
- **纵向** — 从古到今
- **横向** — 从内到外
- **辩证** — A → B → A+B

### 4个避免

1. 避免"伪辩证" — 硬凑正反
2. 避免"全盘否定/肯定" — 一边倒
3. 避免"模板化" — 看到"创新"就套"科技创新"
4. 避免"假大空" — 高大上口号, 无具体内容

### 5个角度

1. **个人成长** — 材料对"我"的启示
2. **社会现实** — 材料对"社会"的启示
3. **国家民族** — 材料对"国家"的启示
4. **人类文明** — 材料对"人类"的启示
5. **文化哲学** — 材料的文化意蕴与哲学内涵

## 维度3: 结构 (Structure)

### 五大经典结构

| 结构 | 适用文体 | 优点 |
|------|----------|------|
| 起承转合 | 议论文 / 演讲稿 | 经典稳妥 |
| 并列式 | 议论文 / 说明文 | 工整有力 |
| 递进式 | 议论文 / 评论文 | 层层深入 |
| 对比式 | 议论文 / 散文 | 鲜明有力 |
| 时空交错 | 散文 / 记叙文 | 富有创意 |

### 段落公式

**开头段 (≤150字)**:
- 公式A: 引用材料 + 概括立意 + 亮明态度
- 公式B: 场景描写 + 引出主题
- 公式C: 名言警句 + 类比联想

**主体段 (200-350字)**:
- 段首: 一句话亮明本段论点
- 段中: 事例1 + 分析 + 事例2 + 分析
- 段末: 回扣论点 + 过渡

**结尾段 (≤150字)**:
- 公式A: 回扣标题 + 名言 + 时代呼吁
- 公式B: 诗意升华 + 青年担当
- 公式C: 场景再造 + 意象呼应

### 五查清单 (Stage 4)

1. 扣题 — 每段是否服务核心立意
2. 平衡 — 素材是否古今中外
3. 层次 — 论证是否递进
4. 详略 — 比例是否合理
5. 收束 — 结尾是否升华

## 维度4: 素材 (Materials)

### 素材等级

| 等级 | 素材类型 | 得分 |
|------|----------|------|
| ★★★★★ | 古今中外四维度 + 时政 | 58-60 |
| ★★★★ | 中外 + 时政 | 54-57 |
| ★★★ | 古代 + 现代 | 48-53 |
| ★★ | 仅古代 | 45以下 |
| ★ | 仅课本/套话 | 40以下 |

### 五大主题素材包

1. **家国情怀** — 林觉民 / 钱学森 / 袁隆平 / 张桂梅
2. **传统文化** — 屈原 / 苏轼 / 曹雪芹 / 梅兰芳
3. **创新奋斗** — 任正非 / 屠呦呦 / 谷爱凌 / 韦东奕
4. **思辨辩证** — 齐桓公三人 / 廉颇蔺相如 / 苏轼豁达
5. **国际视野** — 曼德拉 / 马丁·路德·金 / 一带一路

### 三结合原则

- 古今结合 — 古代人物 + 当代人物
- 中外结合 — 中国故事 + 外国故事
- 大小结合 — 宏大叙事 + 微观细节

## 维度5: 语言 (Language)

### 三大特征

| 维度 | 优秀特征 |
|------|----------|
| 词汇 | 准确、丰富、文白相间 |
| 句式 | 长短结合、整散结合 |
| 修辞 | 比喻/排比/反问/设问综合 |

### 三美

- 音韵美 — 排比 / 对仗 / 双声叠韵
- 意象美 — 借代 / 借喻 / 象征
- 哲理美 — 警句 / 格言 / 箴言

### 七练

1. 练比喻 — 抽象 → 形象
2. 练排比 — 气势
3. 练引用 — 底蕴
4. 练反问 — 力度
5. 练对偶 — 韵味
6. 练短句 — 张力
7. 练动词 — 画面

### 五大避坑

- 生造词 (是酱紫的, yyds)
- 网络热词过度 (绝绝子, 芭比Q了)
- 翻译腔 (作为一个……我认为……)
- 空洞抒情 (多么……啊!)
- 套话 (作为新时代青年, 我们要……)
